Features

Everything you need in one lightweight app.

FORMATS

Wide Format Support

ZIP, TAR, GZ, BZ2, XZ natively. RAR, 7z, CAB, ISO, DEB, RPM and more via patool.

UI

Modern Dark Interface

Clean dark theme with sortable columns, live search, breadcrumb bar and a file-type icon column.

SPEED

Fast and Lightweight

Single-file executable. Launches in under a second. Minimal memory usage on any machine.

OPS

Full File Management

Browse, copy, paste, rename, delete, create folders. Ctrl+A, F2, Delete shortcuts built in.

TEST

Archive Integrity

Test ZIP and TAR archives for corruption with one click before extracting.

CRC32

CRC32 Calculator

Calculate and display CRC32 checksums for any file directly in the file list.

PREMIUM

Premium Upgrade

Multi-threaded compression, no trial screen, priority support via litefile@proton.me.

WINDOWS

Windows Native

Designed for Windows 10/11. Runs as a standalone .exe or from source. No extra dependencies.

SECURE

Secure & Private

No data collection. No cloud storage. All files stay on your machine.
